Tecnología 2024

La placa base, también conocida como placa madre o motherboard, es el componente fundamental en un ordenador que conecta todos los elementos del sistema. Es el soporte físico y el centro de comunicación entre el procesador (CPU), la memoria RAM, los dispositivos de almacenamiento (como discos duros y SSDs), y las tarjetas de expansión (como las de gráficos y sonido). Además, la placa base incluye puertos y conectores para periféricos, como USB y audio. Su diseño determina la compatibilidad con otros componentes y la capacidad de expansión del sistema. Equipadas con circuitos integrados y buses, las placas base facilitan el flujo de datos y la sincronización de los componentes internos. La elección adecuada de una placa base es crucial para el rendimiento y la escalabilidad del ordenador, ya que afecta la capacidad de actualización y las características generales del sistema.

Componentes de la Placa Base

  • Procesador (CPU): El chip central que ejecuta instrucciones y gestiona el procesamiento de datos. Ejemplos incluyen Intel Core i7 o AMD Ryzen 5.
  • Memoria RAM: La memoria de acceso aleatorio que almacena datos temporales para que el procesador pueda acceder a ellos rápidamente. Ejemplos son módulos DDR4 de 16 GB.
  • Chipset: El conjunto de circuitos que gestiona la comunicación entre el procesador, la memoria y otros componentes. Ejemplos incluyen Intel Z590 y AMD B550.
  • Slots de Expansión (PCIe): Ranuras para tarjetas adicionales, como tarjetas gráficas o de sonido. Ejemplos son tarjetas NVIDIA GeForce RTX 3080 o Sound Blaster Z.
  • Conectores SATA: Puertos para conectar discos duros y SSDs. Ejemplos son discos duros Seagate Barracuda o SSDs Samsung 970 EVO.
  • BIOS/UEFI: El firmware que inicia el sistema y gestiona la comunicación entre el hardware y el software. Ejemplos de marcas incluyen AMI y Phoenix.
  • Puertos USB: Conectores para dispositivos periféricos, como ratones, teclados y unidades externas. Ejemplos son USB 3.2 y USB-C.
  • Conectores de Energía: Puertos para conectar la fuente de alimentación del sistema. Ejemplos incluyen conectores ATX de 24 pines y conectores de 8 pines para la CPU.
